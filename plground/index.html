About PLGround

PLGround provides a wiki-like open document repository. Documents are written in Zord and rendered with an in-browser interpreter. Zord is a functional programming language that advocates compositional programming.

Code Repo: https://github.com/yzyzsun/zord

Contributors: Yaozhu Sun and Bruno Oliveira
